# Moved

The DHT partition-healing algorithm specification is now maintained as part
of the `iroh-gossip-rendezvous` crate:

**<https://github.com/swaits/iroh-gossip-rendezvous/blob/main/PROTOCOL.md>**

The bridge pins a specific `iroh-gossip-rendezvous` version; the spec that
matches whatever version you're running is in
`~/.cargo/registry/.../iroh-gossip-rendezvous-*/PROTOCOL.md` or, for path
dependencies, at `../iroh-gossip-rendezvous/PROTOCOL.md`.
